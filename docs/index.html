Hangul Key
Changer

원하는 키를 한영 전환키로 설정하세요

Remap any key to toggle Korean/English input on macOS

Homebrew로 설치 Download

Features

간편하고 강력한 한영 전환

키 리매핑

오른쪽 커맨드, Caps Lock 등 원하는 키를 한영 전환키로 설정할 수 있습니다.

원클릭 설정

활성화 한 번이면 키 매핑과 시스템 단축키가 자동으로 설정됩니다.

재부팅 후에도 유지

LaunchAgent가 설정을 유지하여 앱 실행 없이도 계속 작동합니다.

메뉴 바 컨트롤

상태 표시줄에서 빠르게 접근하고 설정을 관리할 수 있습니다.

Install

설치

Homebrew

$brew install --cask hulryung/tap/hangulkeychanger

Manual Download

Releases 페이지에서 최신 .dmg 파일을 다운로드하고, 앱을 /Applications 폴더로 드래그하세요.

How to Use

사용법

1

키 선택

"변경" 버튼을 클릭해
전환키를 선택하세요

2

활성화

"활성화"를 클릭하고
관리자 비밀번호를 입력하세요

3

완료

선택한 키로
한영 전환이 작동합니다

Uninstall

제거

앱에서 "비활성화"를 클릭하면 모든 설정이 원래대로 복원됩니다. 이후 앱을 삭제하면 됩니다.
